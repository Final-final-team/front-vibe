# 0015. 멤버 검색/필터 + 리뷰 진입 가드 UX 보강

- 날짜: 2026-03-17

## 배경
멤버 수가 증가하면 원하는 대상 찾기가 느리고, 잘못된 taskId로 리뷰 페이지 진입 시 사용자에게 다음 행동이 명확히 보이지 않았다.

## 선택
- Members 화면에 검색(이름/이메일/팀)과 상태 필터(전체/참여중/초대대기/추가확인) 추가
- 필터 결과가 없을 때 빈 상태 메시지 제공
- Task Reviews 화면에 invalid taskId 진입 가드 추가
- review 비어 있을 때 첫 상신 CTA를 직접 노출

## 장점
- 운영자 탐색 속도 개선
- 오류 진입 시 복구 경로가 명확해짐
- 첫 행동(검토 상신) 유도가 강해짐

## 단점
- 상태 관리 코드가 늘어 컴포넌트 복잡도 증가
- 필터 항목이 늘어날수록 추상화 필요 가능

## 후속
- 멤버 필터를 URL query와 동기화해 공유 가능한 뷰로 확장 검토
- 대규모 데이터에서는 서버 필터링 API 도입 검토
